## Critical Architecture Notes

### Middleware Order (matters!)
Registered in `app/bot.py`, execution order (outermost → innermost):
1. ThrottlingMiddleware (disabled in dev)
2. DBSessionMiddleware
3. UserMiddleware
4. BotStatusMiddleware
5. RoleGuardMiddleware

### Router Order (matters!)
Cancel handler (`app/handlers/common/cancel.py`) must be registered FIRST to catch "❌ Отмена" across all FSM states.

### Redis Database Separation
- DB 0 (`REDIS_URL`): flag cache (`bot_enabled`, `buy_enabled`, `sell_enabled`, TTL 30s)
- DB 1 (`ARQ_REDIS_URL`): task queue for ARQ

### SQLAlchemy Enum Naming
Always use explicit `name=` parameter:
```python
# Correct
role: Mapped[RoleEnum] = mapped_column(Enum(RoleEnum, name="user_role"), ...)

# Wrong (generates "roleenum")
role: Mapped[RoleEnum] = mapped_column(Enum(RoleEnum), ...)
```
