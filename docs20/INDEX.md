# Документация проекта — Индекс

## Принцип SSOT (Single Source of Truth)

Каждая сущность описана в **одном** документе. При упоминании в других документах — **ссылка**, не копирование.

| Сущность | Единственный источник |
|----------|----------------------|
| Архитектурный компонент / middleware | `architecture.md` |
| Структура файлов / конфиги | `modules.md` |
| Таблица БД / колонка / enum / миграция | `database.md` |
| FSM пошаговый сценарий | `scenarios.md` |
| Матрица прав по ролям | `roles.md` |
| Технологии и версии | `stack.md` |
| Активные проблемы и риски | `issues.md` |

---

## Карта документов

```
docs/
├── INDEX.md          ← Вы здесь
├── agents.md         # Правила для AI-ассистента
├── architecture.md   # Архитектура, компоненты, middleware [SSOT]
├── database.md       # Схема БД, таблицы, enum [SSOT]
├── issues.md         # Активные проблемы и риски
├── modules.md        # Структура файлов, конфигурации [SSOT]
├── quickstart.md     # Быстрый старт (5 шагов)
├── roles.md          # Матрица прав, меню по ролям [SSOT]
├── scenarios.md      # FSM-сценарии (пошагово) [SSOT]
└── stack.md          # Технологический стек и обоснование
```

## Навигация по задачам

| Что нужно сделать? | Какой документ читать? |
|-------------------|------------------------|
| Запустить проект | `quickstart.md` |
| Понять архитектуру | `architecture.md` |
| Найти нужный файл | `modules.md` |
| Узнать схему таблицы или enum | `database.md` |
| Реализовать FSM-сценарий | `scenarios.md` |
| Проверить права роли | `roles.md` |
| Узнать версию зависимости | `stack.md` |
| Понять активные проблемы | `issues.md` |
| Настроить AI-ассистента | `agents.md` |

## Правила обновления

| Изменение | Обновить файл |
|-----------|--------------|
| Новый middleware / изменение порядка | `architecture.md` |
| Новый файл / конфигурация | `modules.md` |
| Таблица / колонка / enum / миграция | `database.md` |
| FSM-сценарий | `scenarios.md` |
| Права / роли | `roles.md` |
| Зависимость / версия | `stack.md` |
| Обнаружена / устранена проблема | `issues.md` |
